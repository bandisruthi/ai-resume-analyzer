# 🚀 AI Resume Analyzer

A role-based AI Resume Analyzer built using **Flask and NLP** that evaluates resumes and provides skill-based feedback.

---

## 🔥 Features
- 📄 Upload Resume (PDF)
- 🧠 Extract text using NLP
- 🎯 Role-based skill analysis
- 📊 Resume score calculation
- ⚠️ Missing skill suggestions

---

## 🛠 Tech Stack
- Python
- Flask
- PDFMiner
- HTML/CSS
